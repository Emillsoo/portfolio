# 외래 대기안내 연동 배치 (WaitGuideMark)

`Java` · 배치 · 시스템연동

| 한 줄 | 외래 진료 대기 흐름 데이터를 대기안내(표시) 시스템과 연동하는 Java 배치 |
|---|---|
| 역할 | 배치 연동 개발·운영 |
| 핵심 역량 | 대량 처리 배치 설계 · 로그 기반 운영 안정화 |
| 상태 | 운영 중 |

> ⚠️ 기관/화면 식별정보 일반화.

## 개요
외래 진료의 대기 상태를 수집·가공해 대기안내(표시) 시스템으로 전달하는 Java 배치. 대량 처리와 로그 기반 운영 모니터링을 포함한다.

## 핵심 포인트
- 주기 배치로 대기 상태 수집 → 표시 시스템 연동
- 대량 로그 기반 운영 상태 추적
- 진료 대기 흐름의 정합성 유지
