# 사내 OCR API 서비스 (FastAPI + PaddleOCR)

> ⚠️ `.env`(API 키)·내부 경로·개인 샘플 이미지는 포함하지 않습니다.

## 문제
n8n 등 내부 시스템이 이미지→한국어 텍스트 추출을 HTTP로 재사용할 수 있는 사내 OCR이 필요.

## 접근 / 아키텍처
```
FastAPI: /health, /ocr(multipart)
  lifespan 모델 워밍업
  → 동시성 가드(Semaphore 1 + 대기열 한도)
  → 초과 시 429/503 조기거절(추정 대기시간 기반)
  → 다운스케일(폭 1920) → PaddleOCR 추론 → 텍스트 반환
```

## 핵심 안전장치
- 실데이터 추론 27~88초 → n8n 60초 timeout 적체 문제를 **다운스케일 + 조기거절**로 해결(사고 기반 튜닝, 주석에 근거 기록)
- Fail-safe: 크기 10MB·확장자 화이트리스트·빈결과 처리, 선택적 API Key
- 배포: nssm 서비스(부서 표준 패턴)

## 보여주는 역량
과부하 방어를 갖춘 안정적 OCR 마이크로서비스 설계(동시성·조기거절·성능 튜닝).

## 가상 재구현
`impl/` — 동일 동시성 가드/조기거절 로직을 더미 OCR 백엔드로 재현한 FastAPI 데모.
