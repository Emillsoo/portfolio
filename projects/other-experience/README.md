# 그 외 경험 (요약)

포트폴리오 대표/보조로 별도 정리하기엔 규모가 작거나, 직접 구현 범위가 아직 명확히 확정되지 않아 **정직하게 요약만** 남기는 항목들.

> ⚠️ 기관/개인 식별정보는 모두 일반화. 과장 방지를 위해 확정된 범위만 기술.

## 진료실 대기 전광판 호출 흐름
진료실 대기 호출을 전광판에 표시하는 연동 흐름 **설계**. 대기·호출 이벤트를 표시 장치로 전달하는 구조 정리.

## 회의록 자동화 (Next.js + Python STT)
회의 녹음/영상 → STT → 회의록 자동 생성 풀스택 구성(Next.js 프론트 + Python 백엔드). **직접 구현 범위가 명확히 확정되지 않아** 대표/보조에서 제외하고 경험으로만 기재. 회의 내용/참석자 정보는 비공개.

## 새롬 채용/공단입금 수납처리 자동화
채용건·공단입금 수납처리 자동화 **초안 단계**. 반복 수납 업무 자동화 후보로 검토.

## 전산개발위원회 과제 마스터 (프로세스)
분기별 과제(요청서/계획서/기술문서/일정/진행계획)를 관리한 **거버넌스·기획 컨텍스트**. 여러 API·자동화 프로젝트의 상위 관리 틀.
