# 콜프로그램 / 케어메시지 벤더연동 API (Java · EMR)

> ⚠️ [협력업체]명·서브밋·자격증명은 전부 일반화/제거했습니다.

## 문제
EMR과 외부 솔루션(콜프로그램·케어메시지) 간 연동을 위한 EMR 측 API/인터페이스가 필요.

## 접근
- EMR 서브밋/인터페이스 **직접 개발**, 연동 파라미터·응답 규격(REST/SOAP) 정리
- 자격증명 발급·연동 규격 문서화, Redmine 하위일감으로 진행관리

## 사용 기술
Java(EMR 컴포넌트), iBatis SQL, REST/SOAP.

## 보여주는 역량
외부 벤더 시스템과의 표준 연동 인터페이스를 EMR 내부에서 직접 개발.
