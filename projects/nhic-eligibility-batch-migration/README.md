# NHIC 자격조회 배치 서버이관 (인프라 · 폐쇄망)

> ⚠️ 서버/프록시 주소·인증정보는 포함하지 않습니다.

## 문제
재원환자 대상 공단(NHIC) 자격조회 배치를 신규 서버로 이관해야 함. 폐쇄망·방화벽·32bit COM 의존으로 난이도 높음.

## 접근 / 해결
- WebService.dll(**32bit COM**) → 보안 프록시 경유 공단 통신, 결과 DB 저장 구조 이관
- 서버 이관 시 데이터소스를 **DB 직접 → EMR API(.jnu, ex_interface 인증)로 전환**해 방화벽 제약 우회
- 폐쇄망 오프라인 패키지 설치, 방화벽·DNS 디버깅, 작업스케줄러 등록

## 보여주는 역량
폐쇄망/32bit COM/방화벽 등 인프라 제약 하의 배치 서비스 이관·운영 역량.
