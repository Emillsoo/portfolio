# 건강증진팀 사전안내 SMS 자동발송 + 장애복구 (n8n)

> ⚠️ 환자 등록번호/이름 예시·템플릿ID·계정은 포함하지 않습니다. 발송 건수 수치는 사용하지 않습니다.

## 문제
검진 대상자에게 개인맞춤 사전안내문자(유의사항·링크)를 자동 발송. 2026-03 필드명 불일치·응답판정 오류로 미발송 장애 발생.

## 해결 / 아키텍처
- PL/SQL 방식 → **EMR SMS HTTP API 전환(V2.0)**
- 장애 원인: 응답 구조 오판정 → **실제 응답 XML `<sendflag>` 기준으로 판정 수정**
- **DB(발송이력 테이블) 대조 모니터링 신규 구축** — "성공 표시-실제 미발송" 구조적 취약점 제거
- 일시 네트워크 오류(EAI_AGAIN) 대비 Retry, 미발송분 Python 소급 재발송

## 보여주는 역량
발송 결과를 화면 표시가 아닌 **DB 실적과 대조**하는 모니터링 전환 설계.
