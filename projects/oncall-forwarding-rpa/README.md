# 콜당직 착신전환 자동화 (RPA)

> ⚠️ 당직자 실명·휴대폰번호·교환기 관리자 계정은 절대 포함하지 않습니다(전부 제거/합성).

## 문제
전산실 콜당직 교대일마다 교환기(IP-PBX) 착신전환 목적지를 당직자 번호로 수기 변경.

## 접근 / 아키텍처
```
작업스케줄러(교대일 1/6/11/16/21/26일만)
  → RF+Selenium: 교환기 웹 관리페이지 로그인 → DN 조회 → 목적지 변경 → 적용
  → 당직 스케줄(CSV) 기반 번호 변환(외선 접두 처리)
```

## 핵심 기술
Robot Framework + SeleniumLibrary, Chrome headless, iframe/팝업·사설인증서 처리, 폐쇄망 Python.

## 보여주는 역량
주기적 수작업을 스케줄 기반 무인 처리로 전환하는 RPA 설계와 민감정보 취급 원칙.
