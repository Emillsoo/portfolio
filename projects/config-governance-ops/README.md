# 형상관리 거버넌스 & 서버운영 자동화

> ⚠️ 서버 호스트·계정·내부 URL은 포함하지 않습니다.

## 문제
개발물의 변경·배포·승인·롤백이 표준화되지 않으면 운영 사고와 추적 불가로 이어진다.

## 접근
- **GitHub 기반 형상관리 거버넌스** 수립: 변경/배포/승인/롤백 규칙, 트랙 분리
- **서버 운영 규칙 위키** + playwright_server 운영 매뉴얼(등기우편 등 무인 운영 절차)
- 운영 스크립트(Python) 및 롤백 절차 표준화

## 역할
형상관리 관리 책임자로서 거버넌스 제도화.

## 보여주는 역량
개발-배포-운영의 거버넌스 설계와 장애 재발방지의 제도화.
