# 건진 검사장비 수치연동 (청력 · 동맥경화도)

> ⚠️ DSN·IP·스키마/테이블명·환자번호 예시는 포함하지 않습니다.

## 문제
건진 검사장비(청력·동맥경화도) 측정 원본 수치를 EMR에 자동 적재하고, 소비 시스템이 표준 조회로 재사용해야 함.

## 접근 / 아키텍처
```
장비 산출 파일 생성
  → 폴더감시 데몬(device_watcher) 감지
  → 파형/수치 파서
  → EMR 서브밋 적재(게이트웨이)
  → 결과 테이블(세로형 EAV, 자연키 5컬럼) 저장
  → 소비측: Oracle SELECT 표준 조회(마스터 조인)
```

## 핵심 기술 / 설계
- Python watchdog형 데몬, 파형 파서
- **EAV 스키마 설계**(재검사 덮어쓰기), 문자열 값(NR/판정코드) 처리
- 환경 정합(교육/운영), n8n 소비 함정 문서화(대문자 컬럼·executeOnce·0행 가드)

## 보여주는 역량
장비 연동 데몬 + EAV 데이터 모델링 + 표준 조회 인터페이스 설계.
