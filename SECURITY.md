# 보안 및 공개 정책

이 저장소는 의료기관 실무 경험을 **일반화·재구현**하여 공개한 포트폴리오입니다.

## 포함하지 않는 것
- 실제 EMR/HIS 소스코드, 서브밋ID, 테이블/프로시저명, 화면 경로
- 환자 개인정보(성명·주민번호·등록번호·임상데이터) — 모든 예시는 합성
- 기관명·협력업체(벤더) 실명 — [의료기관]/[협력업체]로 일반화
- 계정·비밀번호·API 키·인증서·계좌번호 — 일절 미포함

## 검증
- 게시 전 `security/scan.py`로 비밀·PII 자동 스캔(종료코드 0=클린) 통과
- 커밋 히스토리 오염 방지를 위해 검수 통과 후 클린 커밋으로 시작
